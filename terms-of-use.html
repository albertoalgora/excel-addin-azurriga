2. Descripción del Servicio

El Add-in Azurriga es una herramienta de productividad para Microsoft Excel que permite:

  • Conectarse a un servidor OData para obtener datos financieros
  • Autenticarse mediante nombre de usuario y contraseña
  • Descargar datos de cuentas, flujos de caja y movimientos contables
  • Insertar automáticamente los datos en hojas de Excel con formato apropiado

3. Requisitos del Sistema

Para usar el Add-in, necesita:

  • Microsoft Excel (Office 2016 o posterior, o Microsoft 365)
  • Conexión a Internet
  • Credenciales válidas para acceder al servidor de datos
  • Sistema operativo compatible con Office Add-ins
